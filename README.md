
# Fitness Tracker App – Product Management Case Study

# Overview
This project demonstrates end-to-end product management of a fitness tracker iOS app using Agile Scrum methodology in Jira.

# Problem Statement
Users need a simple way to track fitness activities such as steps, calories, and weight in one place.

# Product Approach
- Defined epics and user stories in Jira
- Prioritized backlog based on MVP
- Planned and executed sprints

# Sprint Execution
### Sprint 1 Goal:
Enable users to register and track steps

# Delivered Features:
- User registration
- Step tracking
- Step dashboard

# Velocity:
~26 story points

# Metrics
- Velocity tracking
- Burndown analysis

# Tools Used
- Jira
- SwiftUI
- Firebase
- CoreML
